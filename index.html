Le Verbe Irrégulier — Être
"To be" — a completely irregular verb, very important!
⚠️ Important Rules for Être
🔹 For nationality and profession → use être only
🔹 For describing someone or something → use être
🔹 Never use être to say someone's name or age
🔊 Tu es indien. 🔊 Vous êtes grand.

CER Verbs — Soft C Rule
Remove -er and add endings · BUT: Nous → c becomes ç to keep soft "s" sound
GER Verbs — Soft G Rule
Nous form adds an extra "e" before "ons" to keep soft "j" sound
È Rule — E + Consonant + ER Verbs
First "e" changes to "è" in Je/Tu/Il/Elle/Ils/Elles · Nous & Vous NEVER change
